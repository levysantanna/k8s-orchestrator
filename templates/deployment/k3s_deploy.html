What This Does

This will:

  1. Connect to your server via SSH
  2. Install k3s using the official installer
  3. Retrieve the kubeconfig
  4. Add the cluster to the orchestrator
  5. Provide the join token for adding workers
Deployment Time

Typically takes 2-5 minutes depending on server speed and network.

SSH Key Format

Supports OpenSSH, RSA, Ed25519 private keys. 

ssh-keygen -t ed25519 -C "k3s-deploy"
